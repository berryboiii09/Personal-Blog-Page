# Personal-Blog-Page
A one-page personal blog website built using HTML, CSS, and JavaScript. The site displays multiple blog posts with interactive controls that let users filter posts by category or search by title, making it easy to browse and find content.

## Key Features:

Responsive layout optimized for desktop and mobile devices

Clean, modern, and easy-to-navigate interface

Buttons to filter blogs by category

Search bar to find blogs by title
